Übertragen von Web-Adressen


1.

Geben Sie die gewünschte URL in der URL Eingabezeile in OpenOffice ein
oder suchen Sie eine URL über die Suchfunktionen in der Hyperlinkleiste.

2.

Markieren Sie die URL, die in der URL Eingabezeile erscheint
(beispielsweise https://www.openoffice.org).

3.

Wählen Sie den Befehl Bearbeiten -> Kopieren, oder drücken Sie STRG+C.

4.

Kehren Sie zu Ihrem Dokument zurück. Danach fügen Sie die URL durch
Bearbeiten -> Einfügen oder per STRG+V ein.

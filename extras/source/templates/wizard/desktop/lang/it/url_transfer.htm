Trasmissione di indirizzi Web


1.

Indicate l'indirizzo (URL) desiderato nel campo di digitazione OpenOffice oppure ricercatene uno tramite le funzioni di ricerca della barra degli hyperlink.

2.

Selezionate l'indirizzo (URL) che appare nell'apposita barra (ad esempio https://www.openoffice.org).

3.

Selezionate poi il comando Modifica -> Copia, oppure CTRL+C.

4.

Ritornate al documento e inserite l'indirizzo (URL) con Modifica -> Incolla oppure con CTRL+V.

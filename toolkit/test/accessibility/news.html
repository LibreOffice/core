News for AWB v1.7.2

Adaptation to recent UAA changes:

  • Introduction of com.sun.star.accessibility.TextSegment structure that is returned by some functions of the XAccessibleText interface.
  • Removal of the COLLAPSED state.
  • Renaming of XAccessibleSelectioni::selectAllAccessible to selectAllAccessibleChildren.

News for AWB v1.7

  • This help window.
  • Integrated relocation of UAA files from drafts to final.
  • Control left click in the graphical view expands the object that has been clicked at.
  • Introduction of split panes for easily changing the size of the three main widgets.
  • The graphical view visualizes the screen size.
  • Removed the "Load" button.
  • Removed the status line.
  • The tree view shows top level nodes only for document windows.
  • New "views" menu that allows you to select the zoom scale of the graphical view.
